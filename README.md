A simple and focused Pomodoro timer for time tracking and productivity. Perfect for study sessions, work intervals, or any task that benefits from structured time management.

## Features

- Fixed 25‑minute countdown timer
- Audible alarm when time is up
- Three simple controls:
  - **Start** – Begins the countdown
  - **Stop** – Pauses the timer
  - **Reset** – Resets the timer back to 25 minutes

## How to Use

1. Open the timer page.
2. Click **Start** to begin a 25‑minute work session.
3. To pause, click **Stop**.
4. To restart the current session, click **Reset** (returns to 25:00).
5. When time runs out, an audible signal will notify you.

## Built With

- HTML, CSS, JavaScript

## Getting Started

Clone the repository and open `index.html` in any modern browser.
